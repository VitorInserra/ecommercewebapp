# siter

#Backend

##Run (see prereqs and installations first)

###Backend
  
  `conda source`

  `cd backend`

  `python3 main.py`
 
 ###Frontend
  
  `npm start`

  

##Prerequisites
1. Python 3.10.2
2. Pip
3. Postgre
4. Conda

##Windows

###Installations

`cd backend siter2`

`conda source`

`pip3 install flask`

`pip3 install flask-login, flask-sqlalchemy, flask-cors`
