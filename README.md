# siter

## Run (see prereqs and installations first)

### Backend
  
  `conda source`

  `cd backend`

  `python3 main.py`

## Prerequisites
1. Python 3.10.2
2. Pip
3. Postgres
4. Conda

### Installations

`pip3 install flask`

`pip3 install flask-login, flask-sqlalchemy, flask-cors`

`pip3 install psycopog2`
